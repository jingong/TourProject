# TourProject
模仿QQ空间，包括前端和后台，主持图片和音乐的上传，记录的删除
前端使用HBuilder编写
后台使用IDEA编写
数据库采用MySQL数据库
服务器为Tomcat

项目完成的功能：
1.发表说说
2.上传图片
3.上传音乐文件
4.下载音乐
5.删除说说
6.查看说说
